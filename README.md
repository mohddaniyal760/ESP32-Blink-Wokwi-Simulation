# ESP32 LED Blink Simulation using Wokwi

A lightweight embedded C project demonstrating a basic LED blinking sequence on an ESP32 microcontroller using the Wokwi simulation framework.

## 🚀 Live Demo
You can interact with and run the live simulation directly in your browser:
👉 [**https://wokwi.com/projects/467893258737809409**]

## 🛠️ Circuit & Components
* **Microcontroller:** ESP32
* **Actuator:** 1x Red LED
* **Passive:** 1x 220Ω Resistor
* **Pin Configuration:** LED connected to GPIO 2

## 📝 Code Overview
The project utilizes standard GPIO manipulation functions:
* `pinMode()` sets GPIO 2 as an output.
* `digitalWrite()` toggles the state between `HIGH` and `LOW`.
* `delay()` introduces a 1000ms interval.
